Программа парсинга сайта www.trendyol.com


Программа настраивается из файла config.json

Подробнее про атрибуты и какие значения рекомендуется ставить ...

Файл config.json - это файл настройки работы программы. В нем есть следующие параметры:
	1. timeout: этот параметр задает максимальное время отклика сайта. если ответа за данный промежуток времени не произойдет. программа завершит работу. Оптимальное значение 10 сек.
	2. perpage: этот параметр задает шаг парсера. То есть если там стоит 1, значит парсер будет качать каждую страницу, если 25 (или др.) то он будет качать каждую 25 страницу. Этот параметр нужен для того чтобы собрать часть информации.
	3. gap: этот параметр задает интервал ожидания. он нужен для того чтобы, после каждой загрузки подождать некоторое время, иначе сервер может забанить. Не рекомендую ставить больше 1.0. ВЛИЯЕТ НА СКОРОСТЬ РАБОТЫ!!!
	4. debug: устанавлевает особый режим. ЛУЧШЕ НЕ ТРОГАТЬ.
	5. key: НЕ ТРОГАТЬ !!!
	6. local: загружать картинки или нет 
	7. headers: заголовки таблицы
	8. api-url: ссылка на api trendyol. НЕ ТРОГАТЬ.
	9. categories: содержит список категорий (НА ЛАТИНСКОМ ЯЗЫКЕ) которые отображаются в URL страницы. Эти категории и будет парсер скачивать. (К справке:
я сам не пробовал но в принципе можно попробовать и спарсить другие данные, для этого нужно заполнить список нужными категориями ОБЯЗАТЕЛЬНО ИЗ ССЫЛКИ ( заходите на сайт кликаете нужную категорию и в верху браузера в строке ссылки, будет что-то типа trendyol.com/.../../...имяраздела+имякатегории/.../... берете имя категории и заполняете им список в config.json categories в формате в котором он уже записан.) Если вы случайно испортите файл config.json просто скопируйте в него все из файла defaul_config.txt
